# zhiwen

    这是在学习jQuery插件时所写，很感谢李炎恢老师所讲的jQuery视频教程。
    
    在学习jQuery时，先看了一本叫做jQuery应用开发实践指南的书，然后看了李老师的视频教程。
感觉看过一遍书看视频的时候就很是容易了，相当于复习功课一样。

    知问前端项目是对知乎前端的一些功能的模拟，当然也并不是和知乎完全一样。在项目中，用到
了很多jQuery插件，算是对jQuery插件的一个的练习和掌握。主要用到了jQuery UI，jquery.cookie
,jquery.form,jquery.migrate,jquery.validate以及一个编辑器插件，项目包含了登录，注册，退出，
提问，发表评论，显示评论等功能。
